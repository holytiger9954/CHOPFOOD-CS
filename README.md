# 허성범 (Heo Sungbeom) - holytiger9954 포트폴리오

## Intro

> CHOP FOOD MES를 설치 없이 실행할 수 있도록 제작한 데스크톱 런처입니다.
> 웹 기반 MES를 일반 프로그램처럼 사용할 수 있도록 패키징하였습니다.
>
> ### 역할
>
> > * C# 기반 데스크톱 런처 개발
> > * WebView2 연동
> > * 배포용 EXE 패키징
> > * 아이콘 및 실행 환경 구성

## CHOP Launcher

> ### MES 데스크톱 실행 프로그램
>
> * 개발기간 : 2026.06.02 ~ 2026.06.03
> * 연계 프로젝트 : CHOP FOOD MES
>
> ### 개발 환경
>
> > * Language : C#
> > * IDE : Visual Studio
> > * OS : Windows 10
>
> ### 기술 스택
>
> > * .NET Framework
> > * WinForms
> > * Microsoft Edge WebView2
>
> ### 주요 기능
>
> > * MES 웹 서비스 실행
> > * 브라우저 UI 제거
> > * 프로그램 형태의 EXE 배포
> > * 커스텀 아이콘 적용
> > * 자동 최대화 실행
> > * WebView2 기반 웹 렌더링
>
> ### 특징
>
> > 사용자는 별도의 브라우저 접속 없이
> > 일반 업무 프로그램처럼 MES를 사용할 수 있습니다.
